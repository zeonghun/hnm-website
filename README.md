# 기능
● 유저는 메뉴와 상품들을 볼 수 있다  
● 유저는 로그인을 할 수 있다  
● 유저는 상품 디테일을 보기 위해 로그인을 해야한다  
● 로그인한 유저는 상품 디테일 정보를 볼 수 있다  
● 유저는 상품을 검색할 수 있다  
● 유저는 로그아웃을 할 수 있다
